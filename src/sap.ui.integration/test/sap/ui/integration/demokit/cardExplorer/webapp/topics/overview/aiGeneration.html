AI Generation

AI Generated Cards

UI Integration Cards are declarative which enables Large Language Models (LLMs) to generate them based on the best practices and recommended patterns.

UI5 MCP Server

The UI5 MCP Server significantly improves the generation of UI Integration Cards. It provides specialized tools designed to assist LLMs in creating new cards from scratch or editing existing ones, while following best practices for card development.

Benefits of using the UI5 MCP Server

Using the UI5 MCP Server to develop UI Integration Cards offers several key advantages:

  • Up-to-date Best Practices: LLMs gain access to the latest recommended APIs and development patterns for UI Integration Cards, ensuring your cards follow current standards.
  • Version Compatibility: Prevents the use of deprecated properties or APIs that are not available in your project's version, reducing errors and technical debt.
  • Efficient Scaffolding: Helps AI agents start new projects faster with established patterns, saving development time and token costs.
  • Card-Type Specific Knowledge: Provides specialized knowledge for each card type. For example, it can suggest the best-fitting chart types for an Analytical card.
  • Cross-Platform Compatibility: Encourages the use of Declarative card types to ensure compatibility with both web and mobile native platforms.

MCP Server Tools

The MCP server provides the following tools for UI Integration Cards development:

  • get_integration_cards_guidelines: Provides dedicated guidelines for the development of UI Integration Cards.
  • create_integration_card: Creates new UI Integration Cards based on specified parameters.
  • run_manifest_validation: Validates the manifest.json file for UI Integration Cards, as well as regular UI5 applications and libraries.

Getting Started

To start using the UI5 MCP Server for generating Integration Cards:

  1. Set up the UI5 MCP Server by following the installation instructions.
  2. Configure your AI assistant to use the MCP server tools.
  3. Use the available tools to create or enhance your UI Integration Cards.

For more information about AI-generated cards and detailed examples, see the SAP Community Blog Post.

How it works

  • An essential part of LLM-driven card generation is presented with the context setting guidelines in the UI Integration Cards Development Guidelines file. They can be reused in other AI tools.
  • A card template helps LLMs to have a good starting point.
  • The UI5 Manifest schema helps validate the results and correct mistakes.
