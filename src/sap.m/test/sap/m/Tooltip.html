Links - mobile long-press behavior

On desktop, hover/keyboard-focus the links to see the tooltip. On a touch device, the tooltip is automatically disabled for <a> elements so the native browser context menu (Copy / Open in New Tab) is preserved on long-press. This applies regardless of the disabledForMobile property (recommended: do not set tooltips on links).
